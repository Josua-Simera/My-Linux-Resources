Running C++ example code (tested and working for Raspi OS and Ubuntu 18)

-	cd to MT_Software_Suite folder previously untarred

-	sudo apt-get install sharutils

-	sudo sh install the mtsdk linux .sh file
	-	sudo ./mtsdk_linux.....sh

-	Do the linking command recommended
	-	sudo ldconfig /usr/local/xsens

-	Install  C++ compiler and others by installing build-essential package:
	-	sudo apt-get install build-essential
	-	sudo apt-get install libusb-1.0-0-dev
	-	cd usr/local/xsens/examples/mtsdk/xda_public_cpp
	-	sudo make HAVE_LIBUSB=1

-	You should end up with two executable files, one for each example code. 

-	Run the "example" executable created, from xda_public_cpp
	-	sudo ./example_name

-	Perhaps you could modify the src_cpp example to output data and then receive arguments to your python program from one of the methods like those in this article below?
	-	How to receive arguments via shell pipe in python?
	-	http://stackoverflow.com/questions/6106437/how-to-receive-arguments-via-shell-pipe-in-python

-	Install python3.5 first
	-	sudo apt-add-repository ppa:deadsnakes/ppa
	-	sudo apt-get update
	-	sudo apt-get install python3.5

-	Need to create python3.5 virtual environment
	-	sudo apt-get install python3-pip
	-	sudo pip3 install virtualenv
	-	virtualenv -p /usr/bin/python3.5 venv

-	Activate virtual environment 
	-	source /venv/bin/activate

-	cd to MT_Software_Suite folder previously untarred
